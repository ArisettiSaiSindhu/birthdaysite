# 🎂 Memories on a Thread – Interactive Birthday Website

An interactive birthday website created as a heartfelt digital gift to celebrate my sister's 22nd birthday. The website presents cherished memories through a beautiful timeline, animated photo gallery, and personalized messages, creating an immersive storytelling experience.

---

## 🌟 Overview

**Memories on a Thread** is a single-page interactive web application that transforms years of memories into a visual journey. Users can explore different stages of life by clicking on photos to reveal personal messages, enjoying smooth animations, decorative effects, and a memorable birthday tribute.

---

- 🎂 Personalized Birthday Celebration

# 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/memories-on-a-thread.git
```

Open the project folder and launch:

```text
akka.html
```

in your preferred web browser.

No additional installation or dependencies are required.
